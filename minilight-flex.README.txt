Description
-----------

MiniLight is a minimal global illumination renderer. See the main MiniLight
readme for a general description.




Installation
------------

### Requirements ###

* Flash player version 9 (probably in a browser)

### Guide ###

* Unpack html and swf files
* maybe put a permission file in: "C:\Documents and Settings\[user]\Application
Data\Macromedia\Flash Player\#Security\FlashPlayerTrust\minilight.cfg"
containing the unpack directory path.




Building
--------

It is an ordinary simple build. There are no special requirements or
dependencies: Everything needed is in the supplied archive (assuming the build
environment and tools are already prepared).

Run the compiler on the mxml, like:
   mxmlc minilight.mxml
